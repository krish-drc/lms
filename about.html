About the Project

This Library Management System is a simple web project developed for practice and internship purposes. It allows managing books, members, and records easily. The system is built using HTML, Bootstrap and can be extended with backend technologies in the future.

This project was created as part of a Web Development Internship to demonstrate front-end design and basic navigation between pages.
